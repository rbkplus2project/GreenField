## How to Get Started ##
  > Fork
  > Clone your fork
  > Fill (.env) file
  > npm install
  > npm start & npm run server
  > Chrome : 127.0.0.1:3001
  > Sign up
  > Sign in
  > Pick a game
  > Enjoy!
